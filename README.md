# Sibuu Sire Court

This folder contains a simple static website for Sibuu Sire Court.

## Local Preview
Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Add it as a remote:
   ```powershell
   git remote add origin https://github.com/<username>/<repo>.git
   ```
3. Push the `main` branch:
   ```powershell
   git push -u origin main
   ```
4. Enable GitHub Pages in the repository settings, using the `main` branch and `/ (root)` folder.

## Deploy to Netlify
1. Create a Netlify account.
2. Drag and drop this folder into Netlify's site deploy area.
3. Or connect your GitHub repository for automatic deploys.
